# Точка Невозврата

Независимое новостное медиа. Свобода слова, честные новости, защита прав человека.

## О проекте

Сайт создан с использованием генератора статических сайтов [Eleventy](https://www.11ty.dev/).  
Контент пишется в Markdown, шаблоны — Nunjucks. Тема — тёмная, с видеофоном и элементами кибер-эстетики.

**Основные разделы:**
- Главная с видео и последними новостями
- Лента публикаций
- О редакции
- Контакты
- Политика / Общество
- Страницы партнёров и проектов

## Технологии

- [Eleventy](https://www.11ty.dev/) — генератор статики
- Nunjucks — шаблонизатор
- HTML5 / CSS3 (адаптивная вёрстка)
- JavaScript (минимально, для интерактива)
- Node.js (среда выполнения)

## Установка и запуск

1. Клонируйте репозиторий:
   \\\ash
   git clone https://github.com/000585/ТОЧКА.git
   cd ТОЧКА
   \\\

2. Установите зависимости:
   \\\ash
   npm install
   \\\

3. Запустите сервер разработки (с автообновлением):
   \\\ash
   npm run serve
   \\\
   Сайт будет доступен по адресу [http://localhost:8080](http://localhost:8080).

4. Для сборки финальной версии (в папку \_site\):
   \\\ash
   npm run build
   \\\
